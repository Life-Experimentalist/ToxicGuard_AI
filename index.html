🛡️ ToxicGuard AI

Real-time toxicity detection and censoring for a safer web experience. Powered by TensorFlow.js and machine learning—100% privacy-focused with local processing.

🔬 Try Live Demo 📂 View on GitHub 💾 Install Extension

Powerful Features

Advanced AI-powered protection that works seamlessly across your browsing experience

🧠

AI-Powered Detection

Uses TensorFlow.js and the @tensorflow-models/toxicity model to detect harmful content with high accuracy across multiple categories including threats, insults, hate speech, and more.

⚡

Real-Time Processing

Instant feedback as you type with debounced input monitoring and efficient DOM observation for dynamic content. Experience seamless protection without lag.

🔒

Privacy First

All processing happens locally in your browser. No data is sent to external servers, ensuring complete privacy and security of your browsing activity.

🎛️

Customizable Settings

Adjust detection sensitivity, enable auto-censoring, and configure toxicity categories to match your preferences. Full control at your fingertips.

🌐

Cross-Browser Support

Works seamlessly across Chrome, Firefox, and Edge with Manifest V3 compatibility and webextension-polyfill for universal compatibility.

🎯

Smart Targeting

Monitors text inputs, textareas, and contenteditable elements with support for dynamic content, shadow DOM, and modern web frameworks.

See It In Action

Experience real-time toxicity detection with visual feedback. The extension provides immediate visual cues when potentially harmful content is detected, making online interactions safer and more positive.

Easy Installation

Get up and running in minutes with our simple installation process

1

Clone Repository

Clone or download the extension from our GitHub repository to get the latest version with all features

2

Enable Developer Mode

Open your browser's extensions page and enable Developer Mode to load unpacked extensions

3

Load Extension

Click "Load unpacked" and select the ToxicGuard AI extension folder from your downloads

4

Start Protecting

The extension is now active! Configure settings from the popup and enjoy safer browsing

Developer Setup (PowerShell)

Quick start guide for developers using PowerShell on Windows

# Clone the repository git clone https://github.com/Life-Experimentalist/ToxicGuard_AI.git cd ToxicGuard_AI
# Optional: Download local TensorFlow assets for offline usage node setup.js
# Load the folder as unpacked extension in your browser:
# Chrome/Edge: chrome://extensions → Load unpacked
# Firefox: about:debugging → This Firefox → Load Temporary Add-on
