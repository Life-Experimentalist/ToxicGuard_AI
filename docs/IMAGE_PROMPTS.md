## Design Guidelines

- **Primary Color:** #6366f1 (Indigo)
- **Secondary Color:** #10b981 (Green for safe indicators)
- **Warning Color:** #ef4444 (Red for toxic indicators)
- **Gradient:** Linear gradient from #6366f1 to #4f46e5
- **Font Style:** Modern, sans-serif (Inter, Roboto, or similar)
- **Icon Style:** Flat design, minimalist, recognizable at small sizes
- **Screenshot Style:** Realistic browser chrome, professional appearance
- **Illustration Style:** Modern vector graphics, clean lines, tech-focused

## Tools Recommended for Generation

- **AI Image Generation:** DALL-E 3, Midjourney, Stable Diffusion XL
- **Vector Graphics:** Figma, Adobe Illustrator, Inkscape
- **Screenshot Tools:** Browser DevTools, Screenshoteer
- **Icon Creation:** Figma, Sketch, Adobe Illustrator
- **Animation:** After Effects, Lottie, CSS animations

---

**Note:** When using AI image generators, you may need to iterate on these prompts and combine multiple generations to achieve the desired result. Always ensure text is readable and icons are clear at their intended display sizes.
